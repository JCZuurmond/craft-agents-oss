Plugin Framework Docs

The Browser Pane is declared in plugin manifest data and activates lazily when the user opens its contributed side panel.

Public API only

The panel receives its webview partition from ctx.webviewPartition, stores the last URL with ctx.storage, and registers itself through ctx.ui.registerSidePanel().

Host enforced boundary

In Electron, main-process webview hardening validates the plugin partition, strips preload scripts, denies permissions, and blocks unsafe navigation.
